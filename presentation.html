Accès sécurisé Appro
Authentification par code PIN pour le service Approvisionnement avant toute confirmation ou modification

  • Session verrouillable à tout moment
  • Lecture seule par défaut sans code
✅
Confirmation 3 états
Chaque PDR peut être confirmé disponible, en rupture (avec délai + justification) ou avec observation
  • ✅ Disponible en stock
  • ❌ Rupture + justification + délai estimé
  • 💬 Observation sans décision de stock
  • Sauvegarde automatique dans Google Sheets
📧
Rappels automatiques PDR
Chaque mercredi à 08h00, des emails automatiques sont envoyés aux services concernés pour les PDR non confirmés
  • 421-MEC → Appro mécanique + CC
  • 421-CHAU → Appro mécanique
  • 421-INST → Appro installation
  • 423-ELEC → Appro électrique
  • 423-REG → Appro instrumentation
  • Réducteurs/pompes → Interchangeable méca

Sécurité

🔐 Architecture sécurisée

Feuille Google Sheets privée
L'accès au Sheets est restreint. Toutes les requêtes passent par Google Apps Script (WEBAPP_URL) avec authentification Google, jamais en accès direct anonyme.
Dépôt GitHub privé
Le code source et les livrables sont dans un dépôt privé. Vercel est configuré pour les builds automatiques depuis GitHub, avec token d'accès sécurisé.
Codes PIN par rôle
4 codes distincts (Admin, Appro, Exécution, CM) contrôlent les actions sensibles. Configurables depuis les paramètres Admin uniquement.
PWA installable hors connexion
L'application est installable sur mobile et PC. Le Service Worker met en cache les ressources statiques pour un fonctionnement offline partiel.
