Module 2 — Planification · PDR

Fonctionnalités PDR

Confirmation sécurisée, rappels automatiques, filtrage par section et notifications en temps réel

🔒
Accès sécurisé Appro
Code PIN requis pour le service Approvisionnement avant toute action. Session verrouillable à tout moment.
  • Lecture seule sans authentification
  • Code configurable depuis paramètres
✅
Confirmation 3 états
Chaque PDR peut être confirmé avec justification, délai estimé ou simple observation.
  • ✅ Disponible en stock
  • ❌ Rupture + justification + délai
  • 💬 Observation sans décision
📧
Rappels automatiques
Chaque mercredi 08h00, emails automatiques aux services pour les PDR non confirmés.
  • 421-MEC / 421-CHAU → Appro mécanique
  • 423-ELEC → Appro électrique
  • 423-REG → Appro instrumentation
  • Réducteurs/pompes → Interchangeable méca
🔔
Notifications push
Après chaque confirmation, notification immédiate au profil Admin sur tous ses appareils connectés.
  • Notification navigateur + inbox in-app
  • Broadcast multi-devices via Sheets

Module 3 — Moteurs Électriques

Workflow d'approbation

De la soumission à l'approbation — notification push immédiate aux profils concernés

📝
Formulaire intelligent
Champs conditionnels selon le type. Anomalies en multi-sélection (Cramé, Vibration, Court-circuit…) + remarque libre.
  • 🔄 Changement · ➕ Pose · 🔧 Réparation
  • 14 types d'anomalies sélectionnables
  • Remarque stockée colonne Q (Sheets)
🛡️
Approbation Admin
L'administrateur visualise toutes les demandes filtrées par statut et approuve ou refuse avec motif obligatoire.
  • ✓ Approuver (sauf réparations)
  • ✗ Refuser avec justification
  • Mise à jour Google Sheets en temps réel
🔍
Contrôle Matériel (CM)
Le profil Contrôle matériel peut saisir le matricule de remplacement et apposer une signature numérique.
  • Accès sécurisé par code CM
  • Signature canvas enregistrée
  • Notification au demandeur
🔔
Notifications multi-profils
L'approbation / refus déclenche des push notifications sur les profils Électrique, Mécanique et Contrôle matériel.
  • ⚡ Électrique · 🔧 Mécanique · 🔍 CM
  • Moteur réparé → notification Électrique
  • Broadcast tous appareils abonnés

Architecture sécurisée

Accès privé, codes PIN par rôle et PWA installable — données protégées à chaque niveau

🔒
Feuille Google Sheets privée
Accès restreint — toutes les requêtes passent par Google Apps Script authentifié. Jamais d'accès anonyme direct.
🗝️
Codes PIN par rôle
4 codes distincts : Admin, Appro (PDR), Exécution (réalisations) et Contrôle Matériel. Configurables depuis les paramètres Admin.
📦
Dépôt GitHub privé
Code source et livrables dans un dépôt privé. Vercel déploie automatiquement à chaque push via token sécurisé.
📱
PWA installable
Installable sur mobile et PC comme une application native. Service Worker pour fonctionnement offline partiel et notifications push.
